# Quartal

Die Dimension &quot;Quartal&quot;zeigt das Quartal an, in dem eine bestimmte Metrik aufgetreten ist. Das erste Dimensionselement ist das erste Quartal im Datumsbereich und das letzte Dimensionselement ist das letzte Quartal im Datumsbereich. Diese Dimension eignet sich ideal für Trendberichte, da sie es Ihnen ermöglicht, Metriken im Zeitverlauf anzuzeigen.

## Diese Dimension mit Daten füllen

Diese Dimension funktioniert bei allen Implementierungen standardmäßig. Wenn eine Report Suite Daten enthält, funktioniert diese Dimension.

## Dimensionselemente

Dimensionselemente umfassen das 3-Monats-Quartal und das Jahr eines bestimmten Datums.
