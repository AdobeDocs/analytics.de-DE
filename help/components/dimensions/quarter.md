# Quartal

Die Dimension „Quartal“ zeigt das Quartal an, in dem eine bestimmte Metrik aufgetreten ist. Das erste Dimensionselement ist das erste Quartal im Datumsbereich und das letzte Dimensionselement ist das letzte Quartal im Datumsbereich. Diese Dimension eignet sich ideal für Trend-Berichte, da sie es Ihnen ermöglicht, Metriken im Zeitverlauf anzuzeigen.

## Füllen dieser Dimension mit Daten

Diese Dimension ist bei allen Implementierungen vorkonfiguriert. Wenn eine Report Suite Daten enthält, funktioniert diese Dimension.

## Dimensionen

Dimensionen beinhalten das 3-Monats-Quartal und das Jahr eines bestimmten Datums.
