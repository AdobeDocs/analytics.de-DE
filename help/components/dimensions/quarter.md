# Quartal

Die Dimension &quot;Quartal&quot;[ ](overview.md) gibt das Quartal an, in dem eine bestimmte Metrik aufgetreten ist. Das erste Dimensionselement ist das erste Quartal im Datumsbereich und das letzte Dimensionselement das letzte Quartal im Datumsbereich. Diese Dimension eignet sich ideal für Trend-Berichte, da sie es Ihnen ermöglicht, Metriken im Zeitverlauf anzuzeigen.

## Füllen dieser Dimension mit Daten

Diese Dimension ist bei allen Implementierungen vorkonfiguriert. Wenn eine Report Suite Daten enthält, funktioniert diese Dimension.

## Dimensionselemente

Die Dimensionselemente umfassen das 3-Monats-Quartal und das Jahr eines bestimmten Datums.
