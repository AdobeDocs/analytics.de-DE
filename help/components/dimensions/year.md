# Jahr

Die Dimension „Jahr“ zeigt das Jahr an, in dem eine bestimmte Metrik aufgetreten ist. Der erste Dimensionselement ist das erste Jahr im Datumsbereich und der letzte Dimensionselement ist das letzte Jahr im Datumsbereich.

## Füllen dieser Dimension mit Daten

Diese Dimension ist bei allen Implementierungen vorkonfiguriert. Wenn eine Report Suite Daten enthält, funktioniert diese Dimension.

## Dimensionen

Dimensionen enthalten ein bestimmtes Jahr, wobei die Daten in Klammern stehen.
