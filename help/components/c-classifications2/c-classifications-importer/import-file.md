# Browser- und FTP-Import

Sobald Sie die gewünschten Daten in die Classification-Vorlagendatei kopiert haben, können Sie diese Datei in die Adobe-Datenerfassungsserver importieren (hochladen).

>[!NOTE] Vor dem Hochladen neuer Daten müssen Sie die vorhandenen Klassifizierungsdaten nicht (mit der Exportfunktion) herunterladen. Sie müssen nur eine neue Datei hochladen, die die neuen Datenzeilen enthält. Diese Methode verbessert die Leistung.

See [General File Structure](/help/components/c-classifications2/c-classifications-importer/c-saint-data-files.md) for important information about the structure and limitations of uploaded files.

Weitere Informationen zur Fehlerbehebung finden Sie unter [Häufige Probleme beim Hochladen von Classifications](https://helpx.adobe.com/de/analytics/kb/common-saint-upload-issues.html).
