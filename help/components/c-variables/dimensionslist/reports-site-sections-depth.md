# Sitebereichstiefe

Verfügbar, wenn die Pfadsetzung aktiviert ist. Klicks pro Seite geben die Anzahl der Seitenansichten an, die innerhalb des angegebenen Sitebereichs der ersten Seitenansicht des Benutzers vorausgingen.
