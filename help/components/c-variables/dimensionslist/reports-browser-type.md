# Browsertypen

Bezeichnet den Typ des vom Benutzer verwendeten Browsers, z. B. Internet Explorer, Mozilla Firefox, Google Chrome usw.

Browsertypen, die noch nicht identifiziert wurden, werden als „Nicht angegeben“ angezeigt.

Beispiel:

| Google |
|---|
| Mozilla |
| Microsoft |
| Apple |
| Opera |
| Nicht angegeben |
| BlackBerry |
| UC Web |
| Sonstige |
| Nokia |
