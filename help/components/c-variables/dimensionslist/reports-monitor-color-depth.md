# Bildschirmfarbtiefe

Zeigt Daten zur Farbtiefe an, die Besucher Ihrer Website auf deren Computer eingestellt haben. Die Farbtiefe beschreibt die Anzahl der Farben, die auf dem Bildschirm angezeigt werden können.
