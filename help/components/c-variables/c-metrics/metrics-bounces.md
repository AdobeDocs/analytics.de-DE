# Absprünge

Ein Besuch, der aus einem einzelnen Server-Aufruf besteht. Zum Beispiel handelt es sich bei einem einzelnen Seitenbesuch um einen Absprung, wenn ein Besucher keine Interaktion mit der Seite führt, bei der Daten zu Adobe gesendet werden, zum Beispiel durch Anklicken eines Links oder eines Videostarts. Wenn mehrere Hits bei einem Besuch empfangen werden, wird kein Absprung gezählt.
