# Kapitelmetriken für Streaming-Mediendienste

Die Kapitelmetriken zu Streaming-Mediendiensten bieten zusätzliche Berichtsfunktionen für Daten, die über Streaming-Mediensammlungsbibliotheken erfasst werden. Für diese Metriken ist das **[!UICONTROL Add-on Adobe Analytics for Streaming Media“]**. Weitere Informationen erhalten Sie von Ihrem Adobe Account Team.

Um diese Metriken zu verwenden, aktivieren Sie **[!UICONTROL Medienkapitel]** unter [[!UICONTROL Medienberichte]](/help/admin/tools/manage-rs/edit-settings/media-management.md) für die Report Suite.

Die folgenden Metriken sind verfügbar:

* [[!UICONTROL Kapitel abgeschlossen]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/chapter-completes)
* [[!UICONTROL Kapitelstarts]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/chapter-starts)
* [[!UICONTROL Besuchszeit für Kapitel]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/chapter-time-spent)

Entsprechende Dimensionen finden [ im Kapitel ](../dimensions/sm-chapters.md)Dimensionen für Streaming-Mediendienste“.
