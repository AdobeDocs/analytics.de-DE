# Absprungrate

Die Metrik &quot;Absprungrate&quot;zeigt das Verhältnis der Besuche, die genau einen Treffer enthielten, im Vergleich zur Anzahl der Besuche auf dieser Seite. Sie können diese Metrik verwenden, um zu verstehen, welche Dimensionswerte die höchste Absprungrate aufweisen, oder um eine aggregierte Gesamtabsprungrate Ihrer Site im Zeitverlauf anzuzeigen.

## Berechnung dieser Metrik

Diese Metrik wird mit der Formel berechnet [`Bounces`](bounces.md)` divided by `[`Entries`](entries.md).
