# Streaming-Mediendienste: Player-Statusverfolgungsmetriken

Streaming-Mediendienste und Player-Statusverfolgungsmetriken bieten zusätzliche Berichtsfunktionen für Daten, die über Streaming-Mediensammlungsbibliotheken erfasst werden. Für diese Metriken ist das **[!UICONTROL Add-on Adobe Analytics for Streaming Media“]**. Weitere Informationen erhalten Sie von Ihrem Adobe Account Team.

Um diese Metriken zu verwenden, aktivieren Sie **[!UICONTROL Player-Status]** Tracking) unter [[!UICONTROL Media Reporting]](/help/admin/tools/manage-rs/edit-settings/media-management.md) für die Report Suite.

Die folgenden Metriken sind verfügbar:

* [[!UICONTROL Von verdeckten Untertiteln betroffene Streams]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-streams-impacted)
* [[!UICONTROL Geschlossene Untertitelanzahl]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-count)
* [[!UICONTROL Gesamtdauer für verdeckte Untertitel]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-total-duration)
* [[!UICONTROL Vom Vollbildmodus betroffene Streams]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-streams-impacted)
* [[!UICONTROL Anzahl der Vollbilder]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-count)
* [[!UICONTROL Gesamtdauer im Vollbildmodus]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-total-duration)
* [[!UICONTROL Von im Fokus betroffene Streams]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-streams-impacted)
* [[!UICONTROL Anzahl der Fokussierungen]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-count)
* [[!UICONTROL Gesamtdauer des Fokus]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-total-duration)
* [[!UICONTROL Von Stummschaltung betroffene Streams]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-streams-impacted)
* [[!UICONTROL Anzahl der Stummschaltungen]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-count)
* [[!UICONTROL Gesamtdauer der Stummschaltung]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-total-duration)
* [[!UICONTROL Vom Bild betroffene Ströme in Bild]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-streams-impacted)
