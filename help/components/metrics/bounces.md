# Absprünge

Die „Metrik[&#x200B; „Bounces“ &#x200B;](overview.md) die Anzahl der Besuche an, die genau einen Treffer enthielten. Sie können diese Metrik verwenden, um das Verhältnis von Absprung- zu Nicht-Absprung-Traffic zu verstehen, die Top-Entrypages zu sehen, die Besucher zum Absprung bewegen, oder um zu sehen, wie die Gesamtanzahl der Absprünge im Zeitverlauf aussieht.

## Berechnung dieser Metrik

Wenn ein Besuch aus genau einem Treffer besteht, wird der Treffer als Absprung gewertet.

>[!NOTE]
>
>Wenn ein Besucher sowohl einen Seitenansichts- als auch einen Linktracking-Aufruf auslöst, wird der Besuch nicht als Absprung gewertet. Erwägen Sie stattdessen die Verwendung von [Einzelseitenbesuche](single-page-visits.md).
