# Mobil-Berichte

Zeigt Informationen zum Zugriff auf Websites von einem Mobilgerät an. Mobil-Berichte verbessern die Identifizierung von Besuchern, indem Daten über Mobilgeräte gesammelt werden, die zur Ansicht Ihrer Site verwendet werden. Mobil-Berichte können den Hersteller und die Funktionen eines Geräts identifizieren, z. B. Bildschirmgrößen, Video-, Audio- und Cookie-Unterstützung und andere Metriken. Mobil-Berichte stehen für alle Report Suites zur Verfügung.

## Mobil-Berichte {#concept_08B95A72D0BA45EDAE9F56D0161EDC8C}

Zeigt Informationen zum Zugriff auf Websites von einem Mobilgerät an. Mobil-Berichte verbessern die Identifizierung von Besuchern, indem Daten über Mobilgeräte gesammelt werden, die zur Ansicht Ihrer Site verwendet werden. Mobil-Berichte können den Hersteller und die Funktionen eines Geräts identifizieren, z. B. Bildschirmgrößen, Video-, Audio- und Cookie-Unterstützung und andere Metriken. Mobil-Berichte stehen für alle Report Suites zur Verfügung.

Um die Identifizierung von Besuchern für Mobilgeräte zu verbessern, müssen Sie sich auf der Liste des Netzbetreibers (weiße Liste) befinden, um Abonnenten-IDs an eine Domäne zu senden. (Sie können den Netzbetreiber im **[!UICONTROL Visitor Profile]** >- **[!UICONTROL Domains]** Bericht Ansicht haben.)

>[!NOTE] Um Berichte auf einem Mobilgerät anzuzeigen, melden Sie sich bei [Experience Cloud](https://login.experiencecloud.adobe.com) an.

Nähere Informationen zur Mobile-Tracking finden Sie auf der [Mobile Analytics](https://www.adobe.com/de/experience-cloud/topics/mobile-analytics.html)-Site.
