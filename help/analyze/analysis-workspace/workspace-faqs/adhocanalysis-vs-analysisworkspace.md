# Analysis Workspace im Vergleich mit Ad Hoc Analysis

Vergleicht die Terminologie und Aufgaben von Ad-hoc-Analysen mit dem Arbeitsbereich für Analysen.

Analyse Workspace integriert einen Großteil der Ad-hoc-Analysen-Funktionen in den Browser-Workflow. Während einige Terminologie und Funktionen zwischen den Produkten unverändert bleiben, gibt es einige neue Begriffe und Ansätze zur Analyse, die in Analyse Workspace eingeführt werden.

Einen technischen Vergleich der wichtigsten Funktionen und Systemanforderungen zwischen diesen beiden Produkten finden Sie [hier](https://marketing.adobe.com/resources/help/en_US/reference/analytics-product-comparison.html).

## Vergleich der Schlüsselbegriffe  {#section_6109406B83B043A18E46D38F130B1D2E}

| Ad Hoc Analysis | Analysis Workspace |
|--- |--- |
| Projekt | Workspace oder Projekt |
| Workspace | Bedienfeld |
| Bericht | Freiformtabelle |
| Diagramme/Diagramme | Visualisierung |
| Hierarchie: Projekt > Workspace > Berichte | Hierarchie: Projekt > Bedienfelder > Tabellen |
| Rang, Trend, Berichtsvorlagen insgesamt | Freiformtabellen-Visualisierung |
| Flussvorlage | Flussvisualisierung |
| Fallout | Fallout-Visualisierung |
