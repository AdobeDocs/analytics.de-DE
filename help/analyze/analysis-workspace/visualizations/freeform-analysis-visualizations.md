# Visualisierungsübersicht

Workspace Angebot eine Reihe von Visualisierungen, mit denen Sie visuelle Darstellungen Ihrer Daten generieren können, wie Balkendiagramme, Ringdiagramme, Histogramme, Liniendiagramme, Karten, Streudiagramme und andere. Jede Visualisierung verfügt über eigene Einstellungen, die Sie verwalten können. Klicken Sie auf den Namen der Visualisierung, um weitere Informationen anzuzeigen.

YouTube-Video: [Visualisierungstypen im Arbeitsbereich](https://www.youtube.com/watch?v=b1zLEywRa6w&amp;index=39&amp;list=PL2tCx83mn7GuNnQdYGOtlyCu0V5mEZ8sS) für Analysen (2:57)

| Name der Visualisierung | Beschreibung |
|---|---|
| [Bereich](/help/analyze/analysis-workspace/visualizations/area.md) | wie ein Liniendiagramm, jedoch mit einem farbigen Bereich unterhalb der Linie. Verwenden Sie ein Flächendiagramm, wenn Sie mehrere Metriken haben und den Bereich visualisieren möchten, der durch den Schnittpunkt von zwei oder mehr Metriken ausgedrückt wird. |
| [Balken](/help/analyze/analysis-workspace/visualizations/bar.md) | Zeigt vertikale Balken an, die verschiedene Werte über eine oder mehrere Metriken hinweg darstellen. |
| [Lineardiagramm](/help/analyze/analysis-workspace/visualizations/bullet-graph.md) | Zeigt, wie ein Wert, an dem Sie interessiert sind, im Vergleich zu anderen Leistungsbereichen (Zielen) oder Messungen dazu steht. |
| [Kohortentabelle](/help/analyze/analysis-workspace/visualizations/cohort-table/cohort-analysis.md) | Eine *`cohort`* ist eine Personengruppe mit gemeinsamen Merkmalen innerhalb eines vorgegebenen Zeitraums. Die Kohortenanalyse ist z. B. dann nützlich, wenn Sie wissen möchten, wie eine Kohorte mit einer Marke interagiert. Sie können problemlos Trend-Änderungen offenlegen und entsprechend reagieren. |
| [Ringdiagramm](/help/analyze/analysis-workspace/visualizations/donut.md) | Ähnlich wie bei einem Kreisdiagramm zeigt diese Visualisierung Daten als Teile oder Segmente eines Ganzen an. |
| [Trichteranalyse](/help/analyze/analysis-workspace/visualizations/fallout/fallout-flow.md) | Fallout-Berichte zeigen, wo Besucher eine Site verlassen haben (wo sie „ausgefallen“ sind) und wo sie eine vorab definierte Folge von Seiten passiert haben (d. h., wo sie verblieben sind). |
| [Fluss](/help/analyze/analysis-workspace/visualizations/c-flow/flow.md) | Zeigt Kundenpfade durch Ihre Websites und Apps an. |
| [Freiformtabelle](/help/analyze/analysis-workspace/visualizations/freeform-table.md) | Eine Freiform-Tabelle ist nicht nur eine Datentabelle, sondern auch eine interaktive Visualisierung. |
| [Histogramm](/help/analyze/analysis-workspace/visualizations/histogram.md) | Ein Histogramm ähnelt einem Balkendiagramm, gruppiert jedoch Zahlen in Bereiche (Behälter). |
| [Horizontaler Balken](/help/analyze/analysis-workspace/visualizations/horizontal-bar.md) | Zeigt horizontale Balken an, die verschiedene Werte aus einer oder mehreren Metriken darstellen. |
| [Linie](/help/analyze/analysis-workspace/visualizations/line.md) | Stellt Metriken mithilfe einer Zeile dar, um anzuzeigen, wie sich die Werte über einen bestimmten Zeitraum ändern. Ein Liniendiagramm kann nur verwendet werden, wenn die Zeit als Dimension verwendet wird. |
| [Landkarte](/help/analyze/analysis-workspace/visualizations/map-visualization.md) | Ermöglicht die Erstellung einer visuellen Zuordnung einer beliebigen Metrik (einschließlich berechneter Metriken). |
| [Streudiagramm](/help/analyze/analysis-workspace/visualizations/scatterplot.md) | Zeigt die Beziehung zwischen Dimensionswerten und bis zu drei Metriken. |
| [Zusammenfassungsnummer](/help/analyze/analysis-workspace/visualizations/summary-number-change.md) | Je nachdem, welche Zelle ausgewählt ist, zeigt diese Visualisierung die Summen und Zusammenfassungen an. |
| [Änderung der Zusammenfassung](/help/analyze/analysis-workspace/visualizations/summary-number-change.md) | Je nachdem, welche Zellen ausgewählt sind, werden bei dieser Visualisierung Zellen miteinander verglichen. |
| [Text](/help/analyze/analysis-workspace/visualizations/text.md) | Sie können benutzerdefinierten Text zu Ihrem Workspace hinzufügen. |
| [Treemap](/help/analyze/analysis-workspace/visualizations/treemap.md) | Zeigt hierarchische (Baumstrukturierte) Daten als Satz verschachtelter Rechtecke an. |
| [Venn](/help/analyze/analysis-workspace/visualizations/venn.md) | Hiermit können Sie bis zu 3 Segmente (aus Komponenten) und eine Metrik ziehen, um ein Venn-Diagramm zu erstellen. |

## Visualisierungsbereich {#section_DC07F032FBEF4046A40F7B95C28DA018}

To display the Visualizations panel, click **[!UICONTROL Visualizations]** in the side panel.

![Schritt Ergebnis](assets/visualizations.png)

Wenn Sie Adobe Analytics verwenden, werden Sie mit den meisten Visualisierungsarten (z. B. Flächen-, Balken-, Donut- und Liniendiagramme) bereits vertraut sein. Analyse Workspace bietet jedoch Visualisierungseinstellungen und viele neue oder einzigartige Visualisierungstypen mit interaktiven Funktionen.

## Visualisierungseinstellungen {#section_D3BB5042A92245D8BF6BCF072C66624B}

Um darauf zuzugreifen, ziehen Sie [!UICONTROL Visualization Settings]eine Visualisierung in den Bereich [!UICONTROL Freeform Panel]und klicken Sie dann auf das [!UICONTROL Visualization Settings] Zahnradsymbol.

>[!IMPORTANT]
>
>Welche Visualisierungseinstellungen sichtbar sind, ist von der jeweiligen Visualisierung abhängig. Nicht alle Einstellungen gelten für alle Visualisierungen. Darüber hinaus werden einige erweiterte Einstellungen **nur** für bestimmte Visualisierungen wie die [Histogrammeinstellungen](/help/analyze/analysis-workspace/visualizations/histogram.md#section_09D774C584864D4CA6B5672DC2927477)angezeigt.

![](assets/visualization_settings.png)

| Einstellung | Beschreibung |
|--- |--- |
| Prozentsatz | Zeigt Werte in Prozent an. |
| 100 % gestapelt | Mit dieser Einstellung für die Visualisierungen „Bereich gestapelt“, „Balken gestapelt“ und „Horizontalbalken gestapelt“ wandeln Sie Diagramme in zu „100 % gestapelte“ Visualisierungen um. Beispiel: ![](assets/stacked_100_percent.png) |
| Legende eingeblendet | Ermöglicht das Ausblenden des Filterdetailtextes für die Visualisierung der Zusammenfassungsnummer/Zusammenfassungsänderung. |
| Maximale Elemente begrenzen | Hiermit können Sie die Anzahl der Elemente einschränken, die in einer Visualisierung angezeigt werden. |
| Y-Achse bei null verankern | Wenn alle im Diagramm dargestellten Werte deutlich über null liegen, wird der untere Teil der Y-Achse standardmäßig nicht-ZERO sein. Wenn Sie dieses Kontrollkästchen aktivieren, wird die Y-Achse auf Null gesetzt (und das Diagramm wird neu gezeichnet). |
| Normalisierung | Erzwingt Metriken für gleiche Anteile. |
| Zwei Achsen anzeigen | Gilt nur, wenn Sie zwei Metriken haben – möglich sind eine Y-Achse links (für die eine Metrik) und eine rechts (für die andere). |
| Anomalien anzeigen | Reichert Liniendiagramme und Freiformtabellen so an, dass sie auch Datenanomalien anzeigen. |

## Symbol „Visualisierung erstellen“ {#section_9C11D9DEDC42413AA53E69A71A509DFC}

If you are not sure which visualization to pick, click the **[!UICONTROL Create Visual]** icon in any table row. Dieses Symbol wird angezeigt, wenn Sie den Mauszeiger über die Tabellenzeile bewegen. Wenn Sie darauf klicken, wird der Arbeitsbereich für Analysen aufgefordert, eine fundierte Einschätzung darüber vorzunehmen, welche Visualisierung am besten zu Ihren Daten passt. Wenn Sie beispielsweise bis zu 3 Segmente ausgewählt haben, wird ein Venn-Diagramm erstellt. Bei mehr als 3 Segmenten wird ein Balkendiagramm erstellt. Bei anderen Datentypen kann ein Liniendiagramm erstellt werden usw.

![](assets/create-visual.png)

## Rechtsklick auf das Visualierungs-/Feldmenü {#section_05B7914D4C9E443F97E2BFFDEC70240C}

Einstellungen, die kontextuell zu einem Diagramm sind, können aufgerufen werden, wenn Sie mit der rechten Maustaste neben einer Visualisierung oder Bereichsüberschrift klicken. Einige oder alle der folgenden Einstellungen sind verfügbar:

![](assets/right-click_menu.png)

| Einstellung | Beschreibung |
|--- |--- |
| Kopierte Visualisierung/Bedienfeld einfügen | Ermöglicht das Einfügen (&quot;Einfügen&quot;) des kopierten Elements an eine andere Stelle im Projekt oder in ein komplett anderes Projekt. |
| Visualisierung kopieren/Bedienfeld | Hiermit können Sie mit der rechten Maustaste auf eine Visualisierung oder ein Bedienfeld klicken und diese kopieren. |
| Duplikat-Visualisierung/Bedienfeld | Erstellt ein exaktes Duplikat der aktuellen Visualisierung, das Sie dann ändern können. |
| Alle Bedienfelder reduzieren | Reduziert alle Projektfenster. |
| Alle Visualisierungen im Bedienfeld reduzieren | Reduziert alle Visualisierungen in diesem Projektbereich. |
| Alle Bedienfelder erweitern | Erweitert alle Projektfenster. |
| Alle Visualisierungen im Bedienfeld erweitern | Erweitert alle Visualisierungen in diesem Projektbereich. |
| Beschreibung bearbeiten | Hinzufügen (oder bearbeiten) Sie eine Textbeschreibung für die Visualisierung/das Bedienfeld. Diese Beschreibung wird in „Projekt“ > „Projektinfo und Einstellungen“ angezeigt. |
| Bedienfeldlink abrufen | Ermöglicht die Anleitung eines Benutzers zu einem bestimmten Bereich in einem Projekt. |
| Link zur Visualisierung abrufen | Ermöglicht Ihnen das Kopieren und Freigeben dieses Links, um andere direkt zu dieser Visualisierung zu senden. Benutzer müssen sich anmelden. |
| Beginn | (Funktioniert für Fluss, Venn, Histogramm) Löscht die Konfiguration für die aktuelle Visualisierung und öffnet ein neues Bedienfeld, in dem Sie sie neu konfigurieren können. |

## Bearbeiten von Legendenbeschriftungen {#section_94F1988CB4B9434BA1D9C6034062C3DE}

Sie können Seriennamen in Visualisierungslegenden umbenennen (Trichteranalyse, Bereich, Bereich gestapelt, Balken, Balken gestapelt, Donut, Histogramm, Horizontalbalken, Horizontaler Balken gestapelt, Linie, Streu und Venn), um Visualisierungen noch benutzerfreundlicher zu gestalten.

Die Bearbeitung von Legenden gilt **nicht** für: Treemap-, Aufzählungs-, Zusammenfassungs- oder Zahlenvisualisierungen, Text-, Freiform-, Histogramm-, Kohorte- oder Flussvisualisierungen.

So bearbeiten Sie z. B. eine Legendenbeschriftung in einem Liniendiagramm

1. Klicken Sie mit der rechten Maustaste auf eine der Legendenbeschriftungen.
1. Klicken Sie auf **[!UICONTROL Edit Label]**.

   ![](assets/edit-label.png)

1. Geben Sie den neuen Beschriftungstext ein.
1. Drücken Sie **[!UICONTROL Enter]** zum Speichern.
