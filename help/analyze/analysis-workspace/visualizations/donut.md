# [!UICONTROL Ringdiagramm] {#donut}

<!-- markdownlint-disable MD034 -->

>[!CONTEXTUALHELP]
>id="workspace_donut_button"
>title="Ringdiagramm"
>abstract="Erstellen Sie eine Ringvisualisierung, um die prozentualen Anteile eines Ganzen zu vergleichen, typischerweise bei einer geringen Anzahl an Werten."

<!-- markdownlint-enable MD034 -->


>[!BEGINSHADEBOX]

_In diesem Artikel wird die Ringdiagramm-Visualisierung in_![AdobeAnalytics](/help/assets/icons/AdobeAnalytics.svg) _&#x200B;**Adobe Analytics**._<br/>_Siehe [Ringdiagramm](https://experienceleague.adobe.com/de/docs/analytics-platform/using/cja-workspace/visualizations/donut)_ für die ![CustomerJourneyAnalytics](/help/assets/icons/CustomerJourneyAnalytics.svg) _&#x200B;**Customer Journey Analytics** Version dieses Artikels._

>[!ENDSHADEBOX]


Ähnlich einem Tortendiagramm zeigt die ![GraphDonut](/help/assets/icons/GraphDonut.svg) **[!UICONTROL Ringdiagramm-Visualisierung]** die Daten als Teile oder Filter eines Ganzen an. Eine Visualisierung vom Typ „Ringdiagramm“ kann für den Vergleich der prozentualen Anteile eines Ganzen verwendet werden, typischerweise bei einer geringen Anzahl an Werten.
