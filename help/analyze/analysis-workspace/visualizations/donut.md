# [!UICONTROL Ringdiagramm] {#donut}

<!-- markdownlint-disable MD034 -->

>[!CONTEXTUALHELP]
>id="workspace_donut_button"
>title="Ringdiagramm"
>abstract="Erstellen Sie eine Ringvisualisierung, um die prozentualen Anteile eines Ganzen zu vergleichen, typischerweise bei einer geringen Anzahl an Werten."

<!-- markdownlint-enable MD034 -->


>[!BEGINSHADEBOX]

*In diesem Artikel wird die Ringidagramm-Visualisierung in **Adobe Analytics**.<br/>Siehe [Ringdiagramm](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-workspace/visualizations/donut) für die **Customer Journey Analytics**-Version dieses Artikels.*

>[!ENDSHADEBOX]

Ähnlich einem Tortendiagramm zeigt diese Visualisierung die Daten als Teile oder Segmente eines Ganzen. Ein [!UICONTROL Ringdiagramm] kann für den Vergleich der prozentualen Anteile eines Ganzen verwendet werden, typischerweise bei einer geringen Anzahl an Werten.
