# Projekt erstellen – Übersicht

**[!UICONTROL Analytics]** > **[!UICONTROL Workspace]**

Sie können ein robustes Analytics-Projekt erstellen, das auf einer beliebigen Kombination von Visualisierungen, Berichtskomponenten und Datentabellen basiert. Dadurch werden viele Funktionen des Tabellenerstellers aus Ad-hoc-Analysen in Analytics integriert.

In Analyse Workspace können Sie Daten auf eine Art und Weise vergleichen und aufteilen, die zuvor nicht möglich war. Konfigurieren Sie z. B. Rangberichte, nehmen Sie sofortige iterative Änderungen an der Abfrage vor und greifen Sie dann auf die Werte auf der Ebene der Berichte zu und bearbeiten Sie sie.

Die Abfrage wird direkt an die Berichte-Engine gesendet. Sie können Änderungen inline vornehmen, ohne andere Berichte aufzurufen, um Ihre Analyse zu erstellen. Die Ergebnisse werden sofort zurückgegeben, ohne dass der Browser aktualisiert wird.

## Workspace -Seite mit Projektliste  {#section_39AA007D7C384F4E869F842F1C7B11F8}

When you first go to **[!UICONTROL Analytics]** > **[!UICONTROL Workspace]**, the page lists all the projects you own or have been granted access to. You can set this page to be your Adobe Analytics landing page by clicking **[!UICONTROL Set as Landing Page]**. (Wenn Sie diese Option nicht sehen, wie im Screenshot unten, ist sie bereits Ihre Landingpage.)

![](assets/sample-project.png)

Die Workspace -Seite mit Projektliste umfasst die folgenden Informationen:

| Element | Beschreibung |
|---|---|
| Projekt  [Vorlagen](/help/analyze/analysis-workspace/build-workspace-project/starter-projects.md) | Sie können diese vorausgefüllten Projektvorlagen unverändert verwenden oder sie an Ihre Anforderungen anpassen (z. | Ansichten-Tutorials | Wechselt zum YouTube-Video zum [Analyse Workspace](https://www.youtube.com/playlist?list=PL2tCx83mn7GuNnQdYGOtlyCu0V5mEZ8sS). |

## Projektinfo und Einstellungen {#section_63773D0B9E4543E88068ECECB9EEB4C6}

**[!UICONTROL Workspace]** > **[!UICONTROL Project]** > **[!UICONTROL Project Info & Settings]**

![](assets/projectinfo.png)

**[!UICONTROL Project Info & Settings]** enthält Informationen auf Projektebene zum derzeit aktiven Projekt.

| Einstellung | Beschreibung |
|---|---|
| Projektname | Der Name des Projekts. Sie können den Namen durch Klicken mit der Dublette bearbeiten. |
| Erstellt von | Name des Projekteigenschafters |
| Zuletzt bearbeitet | Datum der letzten Änderung am Projekt. |
| Tags | Zur einfacheren Kategorisierung werden alle Tags, die auf ein Projekt angewendet werden, Liste. Sie können Projekte auch taggen, während Sie sie speichern. Ansicht der Tags eines Projekts auf der Workspace-Landingpage in der [!UICONTROL Tags] Spalte. |
| Beschreibung | Eine Beschreibung ist nützlich, um den Zweck eines Projekts zu verdeutlichen. Sie können die Beschreibung durch Klicken mit der Dublette bearbeiten. |
| Wiederholte Instanzen im Projekt zählen | Legt fest, ob wiederholte Instanzen in Berichten gezählt werden. Wenn Sie mehrere sequenzielle Werte für dieselbe Variable haben, können Sie diese entweder als eine oder mehrere Instanzen der Variablen zählen. |
| Visualisierungsfarbschema | Sie können das in Workspace verwendete Farbschema ändern, indem Sie eine andere Farbpalette auswählen oder eine eigene Palette angeben. Diese Funktion wirkt sich auf viele Aspekte in Workspace aus, einschließlich der meisten Visualisierungen. |
| Dichte anzeigen | So können Sie mehr Daten auf dem Bildschirm anzeigen, indem Sie den vertikalen Abstand der linken Schiene, Freiformtabellen und Kohortentabellen reduzieren. |
