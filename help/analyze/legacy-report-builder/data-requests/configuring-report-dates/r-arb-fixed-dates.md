# Definitionen des Anforderungs-Assistenten – Feste Datumswerte

{{legacy-arb}}

Definitionen für feste Datumswerte in Report Builder

| Feld | Definition |
|--- |--- |
| Diese Report Suite | Stellt den gespeicherten Datumsbereich nur für die aktuelle Report Suite zur Verfügung. |
| Alle Report Suites | Stellt den gespeicherten Datumsbereich für alle Report Suites zur Verfügung, auf die der angemeldete Benutzer zugreifen kann. |
| Parameternamen eingeben | Geben Sie einen Namen für das gespeicherte Datum ein. Sie können dieses Datum als Vorlage in Report Builder verwenden. Der Name wird immer dann angezeigt, wenn Sie ein gespeichertes Datum anwenden können. Das Ändern des Vorlagennamen wirkt sich nicht auf die Datumskonfiguration aus, die in zuvor erstellten Datenanfragen verwendet wird. |
