# Adobe Analytics-Inhaltspakete für Power BI

{{legacy-arb}}

Adobe Analytics-Inhaltspakete bieten vorgefertigte Power BI-Dashboards und eine Reihe von Power BI-Berichten, die Einblicke in den Traffic Ihrer Site und die Benutzerdimensionen liefern. Sie können das Dashboard und die bereitgestellten Berichte verwenden oder anpassen, um die Informationen hervorzuheben, die Ihnen am wichtigsten sind. Die Daten werden automatisch einmal täglich aktualisiert.

Es sind drei Inhaltspakete verfügbar:

* Adobe Analytics
* Adobe Analytics Mobile App Analytics
* Traffic-Analyse in Adobe Analytics

Weitere Informationen finden Sie unter [Adobe Analytics-Inhaltspaket für Power BI](https://powerbi.microsoft.com/en-us/documentation/powerbi-content-pack-adobe-analytics/).
