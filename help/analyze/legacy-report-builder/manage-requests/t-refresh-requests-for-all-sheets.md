# Anforderungen für alle Arbeitsblätter aktualisieren

{{legacy-arb}}

Sie können Anforderungen in allen Arbeitsblättern oder mehrere Anforderungen gleichzeitig aktualisieren.

Um alle Anfragen zu aktualisieren, benötigen Sie Zugriff auf alle in allen Anfragen verwendeten Report Suites. Wenn Ihre Anmeldeangaben nur zum Zugriff auf eine Teilmenge aller verwendeter Report Suites berechtigen, wird bei der Anmeldung ein Warnhinweis angezeigt, die Sie darüber informiert, dass einige der Anforderungen nicht aktualisiert werden können. Wenn Sie in diesem Fall alle Anfragen aktualisieren, werden die Anfragen, auf die Sie keinen Zugriff haben, nicht aktualisiert.

Wenn Sie mehrere Anforderungen aktualisieren, führt das System eine Erfassung und Priorisierung von Anforderungen entsprechend dem Status der Anforderungen (ursprünglich oder abhängig) durch. Das System zeigt Meldungen über den Aktualisierungsvorgang an. Der Aktualisierungsprozess kann je nach der angeforderten Datenmenge bis zu mehrere Minuten dauern.

So aktualisieren Sie Anforderungen für alle Arbeitsblätter

1. Klicken Sie auf **[!UICONTROL Verwalten]**.
1. Aktivieren Sie **[!UICONTROL Alle Blätter]**.
1. Klicken Sie auf **[!UICONTROL Aktualisieren]**.
1. Geben Sie einen Datumsbereich an und klicken Sie auf **[!UICONTROL OK]**.
