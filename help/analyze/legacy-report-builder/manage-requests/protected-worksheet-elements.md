# Geschützte Elemente in Arbeitsblättern

{{legacy-arb}}

Wenn Sie Zellen zum Schutz Ihres Arbeitsblatts sperren, können Sie keine Anfragen erstellen, bearbeiten oder aktualisieren. Heben Sie einen etwaigen Blatt- oder Zellschutz für das Arbeitsblatt auf, bevor Sie den Anforderungs-Manager verwenden.

Wenn Sie die Anfrage aktualisieren und versuchen, eine Zelle während des Aktualisierungsanforderungsprozesses zu bearbeiten, wird ein Excel-Popup angezeigt, um anzugeben, dass die Tabelle derzeit geschützt ist. Nachdem ein Aktualisierungsanforderungsprozess abgeschlossen ist, können Sie auf die Tabelle zugreifen, und der temporäre Tabellenschutz wird entfernt.
