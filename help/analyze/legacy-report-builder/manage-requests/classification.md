# Klassifizierungen

{{legacy-arb}}

Klassifizierungsaufschlüsselungen werden verwendet, um Analytics-Berichtsdaten verwandten Eigenschaften zuzuordnen. Klassifizierungen können für eine Vielzahl von Zwecken verwendet werden, werden jedoch am häufigsten zur Klassifizierung von Kampagnen-Trackingcodes, sowohl internen als auch externen, und Produkt-IDs verwendet.

Beispielsweise kann ein [!UICONTROL Kampagnen]-Bericht eine [!UICONTROL Manager]-Klassifizierung und eine [!UICONTROL Outlet]-Klassifizierung aufweisen. Diese Klassifizierungen können dann entsprechend einer bestimmten Hierarchie sortiert werden und jede Gruppe von Berichtswerten kann in Untergruppen unterteilt werden.

Der Zugriff auf Klassifizierungen erfolgt auf die gleiche Weise wie der Zugriff auf die anderen Berichte einer Report Suite. Die Dimensionen werden dann innerhalb der Gruppe als Bericht der obersten Ebene angezeigt. Wenn ein Bericht Klassifizierungen aufweist, wird eine Menüuntergruppe erstellt, um den Bericht und alle zugehörigen Klassifizierungen am selben Speicherort zu erfassen. Der Bericht wird als letzter Eintrag in der Liste unter den Klassifizierungen angezeigt.

Siehe [Classifications](/help/components/classifications/classifications-overview.md).
