# Layout – Anforderungs-Assistent: Schritt 2

{{legacy-arb}}

Im Fenster [!UICONTROL Anforderungs-Assistent: Schritt 2] können Sie das Berichtslayout, die Kopfzeilenformatierung und die Dimensionsfilterung für die Ausgabe konfigurieren. Sie können zum Konfigurieren der Anforderung ein Pivot-Layout verwenden. Der Vorgang ähnelt dem Erstellen von Pivottabellen in Excel.

Alternativ können Sie die Zellen, in denen die Berichtsdaten angezeigt werden soll, manuell über die Funktion [Benutzerdefiniertes Layout](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md) auswählen.
