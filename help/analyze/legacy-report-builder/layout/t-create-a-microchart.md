# Microchart erstellen

{{legacy-arb}}

Im benutzerdefinierten Layout können Sie eine Metrik-Spalte in ein Balkendiagramm mit einem zugehörigen Prozentwert oder in einen Prozentwert mit einem zugehörigen Balkendiagramm konvertiert werden. Der Prozentwert ist der Metrikwert oder die Summe für die gesamte nicht gefilterte Anfrage für den Zeitraum.

1. Erstellen Sie eine Anforderung in Excel unter Verwendung des [benutzerdefinierten Layouts](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md).
1. Klicken Sie mit der rechten Maustaste auf die Spalte mit dem Metrikwert.
1. Wählen Sie **[!UICONTROL Ausgabeanzeige ändern]** und geben Sie dann den Microchart-Typ ein, der angezeigt werden soll.

   Report Builder ändert die Anzeige in der Spalte entsprechend Ihrer Auswahl.
