# Erstellen einer Data Warehouse-Anforderung

So erstellen Sie eine Data Warehouse-Anforderung:

1. Navigieren Sie zu [!UICONTROL **Analytics**] > [!UICONTROL **Werkzeuge**] > [!UICONTROL **Data Warehouse**].

1. Konfigurieren Sie auf der Registerkarte Data Warehouse Request die Optionen, die unter [Beschreibungen der Data Warehouse-Anforderungen](/help/export/data-warehouse/data-warehouse.md).

Klicken Sie auf [!UICONTROL **Diesen Bericht anfordern**].
