# Von Data Warehouse bereitgestellte Berichte

>[!NOTE]
>
>Die folgenden Informationen beziehen sich auf FTP- und SFTP-Zieltypen. FTP und SFTP sind alte Zieltypen. Beim Konfigurieren eines Data Warehouse-Berichts sollten Sie einen Cloud-Zieltyp verwenden, der sicherer ist. Weitere Informationen zum Konfigurieren von Cloud-Zieltypen für einen Data Warehouse-Bericht finden Sie unter [Erstellen einer Data Warehouse-Anfrage](/help/export/data-warehouse/create-request/t-dw-create-request.md).

Mithilfe von Data Warehouse können Sie individuelle Fragen stellen und Berichte anfordern, die darauf basierende erweiterte Datenbeziehungen aus Rohdaten enthalten.

Der FTP-Versand ist in diesem Fall als Teil der Einrichtung [!UICONTROL Erweiterte Versandoptionen] verfügbar. Wenn Sie FTP verwenden, benötigen Sie ein vorhandenes internes FTP-Konto oder ein Adobe FTP-Konto. Adobe richtet das FTP-Konto nicht automatisch über die Data Warehouse-Oberfläche ein. Die FTP-Bereitstellung sollte sofort erfolgen, je nach Dateigröße und Anzahl der verarbeiteten Berichte.

Informationen zu FTP-Beschränkungen und zur Datenaufbewahrung finden Sie unter [FTP-Beschränkungen und Datenaufbewahrung](/help/export/ftp-and-sftp/ftp-limits.md).
