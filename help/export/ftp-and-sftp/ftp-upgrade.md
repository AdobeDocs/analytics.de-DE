# Aktualisieren von Adobe FTP- und SFTP-Servern

Bei Upgrades der Adobe-FTP- und -SFTP-Server können neue Konfigurationen implementiert werden, die sich auf Automatisierungsskripte auswirken. Diese werden häufig verwendet, um Daten regelmäßig herunterzuladen oder hochzuladen, und basieren häufig auf bestimmten Server-Einstellungen.

Beispielsweise wird ein neuer Erfolgsstatus eingeführt, der sich auf ein Skript auswirkt, das den Erfolgsstatus zum Trigger einer bestimmten Aktion verwendet. Adobe benachrichtigt Benutzer proaktiv über solche Konfigurationsänderungen. Wenn Adobe Sie darüber informiert, dass ein FTP-Server-Upgrade bevorsteht, überprüfen Sie Ihre Automatisierungsskripte, um sicherzustellen, dass sie weiterhin ordnungsgemäß funktionieren.
