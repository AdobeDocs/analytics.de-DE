# FTP- und SFTP-Beschränkungen und Datenaufbewahrung

Alle Adobe FTP- und SFTP-Konten verfügen über eine Datenspeichergrenze von 100 GB. Dateien werden in diesen FTP-Konten 90 Tage lang aufbewahrt.

## Beschränkungen für gleichzeitige Verbindungen

* Pro Quell-IP höchstens 50 gleichzeitige Verbindungen zum Server
* Pro Quell-IP in 60 Sekunden höchstens 20 neue Verbindungen zum Server
