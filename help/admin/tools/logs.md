# Protokolle

Protokolldateien, die anzeigen, wann sich Benutzende angemeldet haben, was genutzt und worauf zugegriffen wurde, sowie Report Suites und Admin-Änderungen.

**[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Alle Administratoren]** > **[!UICONTROL Protokolle]**

## Admin-Protokoll {#section_8ADE8A7204A8401C968ABC20AECA381D}

Das Admin-Protokoll enthält alle Änderungen, die von Administratoren in den Admin-Tools vorgenommen wurden. Das Protokoll bietet ein Gateway zu benutzerdefinierten Berichten aus jedem der drei Protokolle. Sie können über einen bestimmten Datumsbereich nach Ereignissen suchen, die Ihren ausgewählten Kriterien entsprechen.

## Nutzungs- und Zugriffsprotokoll {#section_6FBAF92D9EA244809C45A78A2F0A7232}

Mit [!UICONTROL Nutzungs- und Zugriffsprotokoll] können Sie die Berichtsnutzung auf Benutzerkontoebene bewerten. Beispielsweise werden darin Aktionen zum Öffnen, Erstellen, Aktualisieren, Aufheben der Freigabe und Löschen in Analysis Workspace verfolgt. Das bietet einen besseren Überblick darüber, wer Workspace wie häufig nutzt.

| Element | Beschreibung |
|---|---|
| Datumsbereich | Geben Sie einen Datumsbereichsfilter an. Sie können ein Datum manuell im Format JJJJ-MM-TT eingeben oder auf das Kalendersymbol klicken, um ein Datum auszuwählen. |
| Anmelden | Filtern Sie das Protokoll nach Benutzername. |
| IP | Filtern Sie das Protokoll nach einer IP-Adresse. |
| Report Suite | Filtern Sie das Protokoll nach einer bestimmten Report Suite-ID. |
| Ereignistyp | Filtern Sie das Protokoll nach einem Ereignistyp. Wählen Sie einen Ereignistyp aus der Dropdown-Liste aus. Siehe die vollständige Liste der Ereignistypen unten. |
| Ereignis | Filtern Sie das Protokoll nach einem Wort oder einer Phrase in der Ereignisbeschreibung. |
| Bericht herunterladen | Exportiert den Inhalt des [!UICONTROL Nutzungs- und Zugriffsprotokolls] in eine durch Tabulatoren getrennte Datei. |

## Report Suite-Änderungsprotokoll {#section_3864966639414BBEA871F4D0352F56B6}

Das Änderungsprotokoll der Report Suite zeigt Änderungen an, die außerhalb von Admin an Ihren Report Suites vorgenommen wurden.

Zu den Tools, die eine Report Suite von außerhalb der [!UICONTROL Admin-Tools“ ändern können] gehören:

* In einem Webbrowser durchgeführte Klassifizierungs-Uploads (Über FTP vorgenommene Klassifizierungs-Uploads sind nicht im Änderungsprotokoll enthalten)
* Änderungen in früheren Versionen.
* Von einem Kundenbetreuer oder der Kundenunterstützung mit internen Tools vorgenommene Änderungen

| Element | Beschreibung |
|---|---|
| Datumsbereich | Geben Sie einen Datumsbereichsfilter an. Sie können ein Datum manuell im Format JJJJ-MM-TT eingeben oder auf das Kalendersymbol klicken, um ein Datum auszuwählen. |
| Firma | Filtern Sie das Protokoll nach Firmenname. |
| Anmelden | Filtern Sie das Protokoll nach Benutzername. |
| IP | Filtern Sie das Protokoll nach einer IP-Adresse. |
| Ereignis | Filtern Sie das Protokoll nach einem Wort oder einer Phrase in der Ereignisbeschreibung. |
| Bericht herunterladen | Exportiert den Inhalt des [!UICONTROL Nutzungs- und Zugriffsprotokolls] in eine durch Tabulatoren getrennte Datei. |
