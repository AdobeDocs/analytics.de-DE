# Web-Services

Die Web Services APIs bieten Programmierungszugriff auf Marketing-Berichte und andere Suite-Services, mit denen Sie die Funktionalität der Analytics-Oberfläche duplizieren und erweitern können.

**[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Alle Administratoren]** > **[!UICONTROL Unternehmenseinstellungen]** > **[!UICONTROL Web-Services]** oder **[!UICONTROL API-Zugriff]**


## Analytics 2.0 APIs

Für den Zugriff auf die Analytics 2.0-APIs benötigen Sie die globale Unternehmens-ID Ihres Analytics-Unternehmens. Die globale Unternehmens-ID finden Sie in **fett** oben im Abschnitt [!UICONTROL API-Zugriff]. Im Folgenden finden Sie ein Beispiel: „Die globale Unternehmens-ID für das Analytics-Unternehmen, bei dem Sie derzeit angemeldet sind, lautet **adobe1234**.“

## Legacy-Web-Dienste verwalten (eingestellt)

Sie können in der [Adobe Admin Console](https://helpx.adobe.com/de/enterprise/using/admin-console.html) den Benutzern Berechtigungen erteilen, die Zugriff auf die Webservices-APIs benötigen.

## WSDL: Laden Sie die WSDL der Web-Services-API für Entwickler von Web-Services herunter.

Besuchen Sie [Adobe Developer](https://developer.adobe.com/analytics-apis/docs/2.0/), wo Sie Dokumentation, Beispiel-Code und Foren zu den Web-Services-APIs finden. Klicken Sie auf Übersicht über Web-Services-APIs , um weitere Informationen zu erhalten.

## Filteroptionen

Wenn Sie SOAP verwenden und Ihr XML-Parser Probleme mit unzulässigen oder ungültigen Zeichen in Antworten auf Webservices-API-Aufrufe hat, wählen Sie eine oder beide der folgenden Optionen aus, damit Analytics die Antwortausgabe automatisch filtert. In der Regel ist dies ein Problem, das nur bei Doppelbyte-Sprachen (Japanisch, Chinesisch, Koreanisch) auftritt.

## API-Zugriffsinformationen

Anzeigen von Web-Services-Zugriffsinformationen nach Benutzer. Diese Tabelle enthält den Webservices-Benutzernamen und das gemeinsame Geheimnis, die Benutzerinnen und Benutzer im Rahmen des Authentifizierungsprozesses beim Aufrufen von Webservices verwenden müssen.

## Token-Nutzung

Zeigt Informationen zur Anzahl der Web-Services-Token an, die von Ihrem Unternehmen im aktuellen Kalendermonat verwendet wurden.
