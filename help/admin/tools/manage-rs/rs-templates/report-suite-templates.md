# Übersicht über Report Suite-Vorlagen

Mit Report Suite-Vorlagen werden die grundlegendsten Einstellungen für verschiedene Arten von Report Suites konfiguriert.

Um beim Erstellen einer neuen Report Suite Zeit zu sparen, können Sie eine Vorlage auswählen, die der von Ihnen entworfenen Report Suite-Konfiguration ähnelt. Die Auswahl einer Report Suite-Vorlage schränkt Ihre Implementierung in keiner Weise ein. Jede Konfiguration, die von einer Report Suite-Vorlage vorgenommen wird, kann nach der Erstellung der Report Suite geändert werden.
