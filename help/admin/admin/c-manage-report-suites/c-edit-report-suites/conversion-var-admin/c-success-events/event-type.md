# Ereignistyp ändern

Sie können den Ereignistyp ändern (also vom Typ „Zähler“, „Numerisch“ oder „Währung“ zu einem anderen Typ wechseln), ohne dass der Zugriff auf die bisher erfassten Daten verloren geht.
