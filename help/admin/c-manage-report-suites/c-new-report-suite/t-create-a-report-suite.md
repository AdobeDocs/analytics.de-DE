# Report Suite erstellen

Beschreibende Schritte zum Erstellen einer Report Suite und zum Kopieren der Einstellungen einer Report Suite in eine neue.

1. Click **[!UICONTROL Analytics]** &gt; **[!UICONTROL Admin]** &gt; **[!UICONTROL Report Suites]**.
1. Wählen Sie eine Report Suite aus.
1. Click **[!UICONTROL Create New]** &gt; **[!UICONTROL Report Suite]**.
1. To copy a report suite's settings, in the template list, select either a predefined template or an existing report suite to use as a [template](../../../admin/c-manage-report-suites/c-report-suite-templates/report-suite-templates.md#concept_539B8C5468424D31B55FC6EB39C5061F).

   >[!NOTE]
   >
   >Es können nur Einstellungen kopiert werden, nicht die Daten. Wenn die Kundenunterstützung die Einstellungen kopiert, müssen Sie den Haftungsausschluss der Kundenunterstützung bezüglich der involvierten Risiken schriftlich bestätigen. See [Settings not copied from a source report suite](../../../admin/c-manage-report-suites/c-new-report-suite/settings-not-copied-from-rs.md#concept_DF1E80ABA99B444C818CED9BC5AC2410) for more information.

1. Fill in the fields described in [New Report Suite](../../../admin/c-manage-report-suites/c-new-report-suite/new-report-suite.md#concept_3CBBE252279C43069030EFBBD7D26376).
1. Klicken Sie auf **[!UICONTROL Report Suite erstellen]**.
