# Report Builder-Berichte

Verwalten Sie Report Builder zugewiesene Lizenzen.

Jeder einem Report Builder zugewiesenen Lizenz werden automatisch zehn geplante Aufgaben zugewiesen. Die Anzahl von Aufgaben kann geändert und die Aufgaben können neu zugeteilt werden. Geplante Aufgaben können nicht die Anzahl an verfügbaren Aufgaben pro Benutzer überschreiten. Aufgaben, die diese Anzahl überschreiten, werden nicht ausgeführt.

Sie können Report Buildern Zugriff auf [!DNL Analytics] Benutzer durch Verwaltung von Produktprofilen in [Adobe Admin Console](/help/admin/admin-console/home.md).
