# In den Produktprofilberechtigungen aufgeführte Report Suite Tools

Verwenden Sie diese Seite, um jedes Berechtigungselement unter Report Suite Tools zu verstehen.

| Berechtigungselement für Report Suite Tools | Beschreibung |
|------|------|
| Data Warehouse | Nicht verwendet. Verwenden Sie stattdessen "Custom Data Warehouse-Bericht" . |
| SiteCatalyst | Nicht verwendet. Verwenden Sie stattdessen den Analysis Workspace-Zugriff. |
| Report Suites (Lesen) | Nicht verwendet. |
| Report Suites (Schreiben) | Nicht verwendet. |
| Traffic-Management | Nicht verwendet. |
| Report Suite-Verwaltung | Gewährt Zugriff auf den Report Suite Manager, erlaubt jedoch keine Änderungen. |
| Kontozusammenfassung | Gewährt Zugriff auf Allgemein &gt; Allgemeine Kontoeinstellungen im Report Suite Manager. |
| URL-Filter | Gewährt Zugriff auf Allgemein &gt; Interne URL-Filter im Report Suite Manager. |
| Benutzerdefinierter Kalender | Gewährt Zugriff auf Allgemein &gt; Kalender im Report Suite Manager. |
| Gebührenpflichtige Suche | Gewährt Zugriff auf Allgemein &gt; Gebührenpflichtige Sucherkennung im Report Suite Manager. |
| Menüanpassung | Gewährt Zugriff auf Allgemein &gt; Menüs anpassen im Report Suite Manager. |
| Konfiguration für Echtzeitberichte | Gewährt Zugriff auf Echtzeit im Report Suite Manager. Verwenden Sie das Berechtigungselement "realtime-Bericht" , um den Zugriff auf den Bericht selbst zu gewähren. |
| Video-Einstellungen | Nicht verwendet. |
| Traffic-Variablen | Gewährt Zugriff auf Traffic &gt; Traffic-Variablen im Report Suite Manager. |
| Kanäle | Gewährt Zugriff auf Marketingkanäle, einschließlich Marketingkanal-Manager, Marketingkanal-Verarbeitungsregeln und Marketingkanal-Ablauf im Report Suite Manager. |
| Kosten | Gewährt Zugriff auf Marketingkanäle &gt; Marketingkanal-Kosten im Report Suite Manager. |
| Konversionsvariablen | Gewährt Zugriff auf Konversion &gt; Konversionsvariablen im Report Suite Manager. |
| Suchmethoden | Gewährt Zugriff auf Konversion &gt; Suchmethoden im Report Suite Manager. |
| Unique Visitor | Gewährt Zugriff auf Konversion &gt; Unique Visitor-Variable. Wird normalerweise nicht in modernen Implementierungen verwendet. |
| Erfolgsereignisse | Gewährt Zugriff auf Konversion &gt; Erfolgsereignisse im Report Suite Manager. |
| Listenvariablen | Gewährt Zugriff auf Konversion &gt; Listenvariablen im Report Suite Manager. |
| Standardmetriken | Gewährt Zugriff auf individuelle Report Suite-Einstellungen &gt; Standardmetriken im Report Suite Manager. |
| Verarbeitungsregeln | Gewährt Zugriff auf Allgemein &gt; Verarbeitungsregeln und Allgemein &gt; Bot-Regeln im Report Suite Manager. |
| Advertising Analytics-Konfiguration | Nicht verwendet. |
| Anomalieerkennung | Gewährt Zugriff auf Anomalieerkennung im Analysis Workspace. |
| Beitragsanalyse | Gewährt Zugriff auf das Menü "Beitragsanalyse ausführen" im Analysis Workspace. |
| Kanalbericht | Ermöglicht Zugriff auf den Marketingkanäle &gt; Kanal-Übersichtsbericht in Reports &amp; Analysen. |
| RealTime Bericht | Gewährt Zugriff auf Site-Metriken &gt; Echtzeit in Reports &amp; Analysen. Verwenden Sie das Berechtigungselement "Konfiguration von Echtzeit-Berichten" , um den Zugriff auf die Einrichtung des Berichts zu gewähren. |
| Bot-Seiten | Ermöglicht Zugriff auf Site-Metriken &gt; Bots &gt; Bot Pages-Berichte in Reports &amp; Analysen. |
| Bots | Gewährt Zugriff auf Site-Metriken &gt; Bots &gt; Bots-Bericht in Reports &amp; Analysen. |
| Datenfeed-Manager | Gewährt Zugriff auf Admin &gt; Data Feeds in der Navigation in Analytics. |
| Benutzerspezifischer Data Warehouse-Bericht | Gewährt Zugriff auf Tools &gt; Data Warehouse in der Navigation in Analytics. |
| Rückkehrende Besucher pro Tag | Gewährt Zugriff auf Besuchertreue &gt; Rückkehrende Besucher pro Tag in Reports &amp; Analysen. |
| Data Sources-Manager | Gewährt Zugriff auf Admin &gt; Datenquellen in der Navigation in Analytics. Erfordert das Berechtigungselement "Report Suite Mgmt" . |
| Nach IP-Adresse ausschließen | Gewährt Zugriff auf Admin &gt; Nach IP ausschließen in der Navigation in Analytics. |
| Letzte 100 Besucher | Nicht verwendet. |
| Veraltete ClickMap | Nicht verwendet. Verwenden Sie stattdessen Activity Map unter Analytics Tools. |
| Ältere Clickmap-Installation | Nicht verwendet. Verwenden Sie stattdessen Activity Map unter Analytics Tools. |
| Rückkehrende Besucher | Gewährt Zugriff auf Besuchertreue &gt; Rückkehrende Besuche in Reports &amp; Analysen. |
| Übersichtsbericht für Mobile Apps | Nicht verwendet. |
| Classifications | Gewährt Zugriff auf alle Klassifizierungseinstellungen im Report Suite Manager. This permission item includes: <br>Traffic &gt; Traffic Classifications<br>Conversion &gt; Conversion Classifications<br>Conversion &gt; Classification Hierarchies<br>Marketing Channels &gt; Marketing Channel Classifications<br>App Management &gt; App Classifications<br>Media Management &gt; Media Classifications<br>AEM &gt; AEM Assets Classifications<br>Adobe Campaign &gt; Adobe Classifications |
| Meine empfohlenen Berichte | Gewährt Zugriff auf Site-Metriken &gt; Meine empfohlenen Berichte in Reports &amp; Analysen. |
| Firmenzusammenfassungs-Reportlet | Nicht verwendet. |
| Bild | Nicht verwendet. |
| KPI/Kennzahlen-Reportlet | Nicht verwendet. |
| Report Suite insgesamt | Nicht verwendet. |
| Text-Reportlet | Nicht verwendet. |
| Gebrauchszusammenfassungs-Kurzbericht | Nicht verwendet. |
| Webquellen | Nicht verwendet. |
| Social | Nicht verwendet. |
